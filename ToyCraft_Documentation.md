## 1. Problem Statement
Toy manufacturers operate in a competitive market where understanding customer demand, seasonal trends, and regional preferences is critical. The goal of this project is to analyze toy sales data using Tableau to generate actionable insights that support strategic decision-making.

## 2. Dataset Overview
The dataset contains information related to toy sales, including:
- Date / Time period
- Product category
- Sales amount
- Region
- Customer demographics

This data enables analysis of trends, preferences, and performance across different dimensions.

## 3. Methodology
The following steps were followed:
1. Data preparation and cleaning.
2. Importing dataset into Tableau.
3. Creating visualizations such as line charts, bar charts, and regional comparisons.
4. Building an interactive dashboard.
5. Interpreting results to derive insights.

## 4. Dashboard Walkthrough
The dashboard includes:
- Sales trend analysis over time.
- Category performance comparison.
- Regional performance visualization.
- Interactive filters for deeper exploration.

Users can interact with filters to explore specific regions, categories, or time periods.

## 5. Key Insights
- Sales increase significantly during holiday seasons.
- Certain categories perform better among specific customer groups.
- Regional differences highlight opportunities for targeted marketing.

## 6. Recommendations
- Increase production before peak demand periods.
- Focus marketing on high-performing categories.
- Optimize inventory distribution across regions.

## 7. Conclusion
This project demonstrates how data visualization using Tableau can help organizations understand market dynamics and make data-driven decisions to improve performance and competitiveness.
